GW Senior CS Design 2014

Alexey Strakovsky: Physics Exercise Generation and Simulation Interface

Physics Exercise Generation and Simulation Interface:The purpose of this project is to create a system which will present students with randomly generated physics problems to solve. These problems would be based on a specific subject and/or equation entered by the student, which the student would like to practice. The system will make use of a sentence parser in the generation of word problems, and a physics engine to simulate both problems and possible solutions as a visual aid. The system will be based on a Java application, which will tie all components together and allow the user to interact with the system.

Bio:I am Alexey Strakovsky, a GWU SEAS undergraduate student seeking a B.S. in Computer Science. This website is for my senior design project. My interests include programming and technology. I hope that my senior design project results in something that can actually be useful to people. I want to be able to produce a tool that helps students to understand physics.
