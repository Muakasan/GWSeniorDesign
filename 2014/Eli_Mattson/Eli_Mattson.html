Bastion: is a solution designed for the average user to browse the web securely. Using sandboxing, a method designed isolate executing code away from the actual root system, Bastion eliminates threats to mobile phones like trojans and other malware. All malicious code gets executed in a temporary sandbox in the cloud away from the device.

Bio: Student, Sailor, CS Engineer with passion for security. I will be graduating GWU and heading to California to work at Shape Security, a security startup.
