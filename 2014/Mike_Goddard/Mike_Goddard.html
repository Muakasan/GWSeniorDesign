GW Senior CS Design 2014

Mike Goddard: Coding Bootcamp

Coding Bootcamp: My project is a training tool to help new programming students learn to code. The project measures the student's programming ability and then dynamically and randomly creates problems for the student to solve. Instead of asking the student to write code, Coding Boot Camp instead provides the student with a finished or nearly finished program, and asks them to follow along and figure out what it is doing. This allows the student to practise without getting overly frustrated with syntax and compilation. The problems are comprised of randomly generated code in a custom built, easy to learn pseudo-code like language.

Bio: I am currently a senior at George Washington University in Washington D.C. I am working towards a Bachelor of Science in Computer Science. After graduation, I intend to live in Boston, MA, where I have recently received a job offer from a local software company.
