GW Senior CS Design 2014

Bianca Lott: E-Quote: A Stock-Market Analyzer

E-Quote: a Stock-Market Analyzer is a project which allows users to see different information pertaining to stock-markets in an organized and readable fashion. For this project, a query language is designed and a parser using ANTLR has been created. The query language and parser will interpret and output information pertaining to different stock markets. A SQL database is also used to help display output of this project to the users.

Bio: I am graduating with a B.S. in Computer Science with minors in History and French Language and Literature. My specific interest in Computer Science is Database Management and Security. I currently work at the George Washington University Libraries where I gained experience working on a digitization project for IMLS (Institute of Museum and Library Services). In my spare time, I like to read, watch movies, cook new recipes and travel.
