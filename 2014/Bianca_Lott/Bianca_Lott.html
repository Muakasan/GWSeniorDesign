Bianca Lott: E-Quote

E-Quote:

This project is called E-Quote, a Stock Market Analyzer. It will allow users to see different information pertaining to stock-markets in an organized and readable fashion. For this project, a query language will be designed and an interpreter will be created. The query language and interpreter will intrepret and output information pertaining to different stock markets using a statistical package influenced by SAS. A database will also be used to display output of this project.
