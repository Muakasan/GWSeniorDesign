Caroline Jung: Network Traffic Monitor

Network Traffic Monitor:

The Network Traffic Monitor is an application that provides utilities and information to assist in the management and analysis of data in a local area network. It provides users with information about different devices on the network, packets captured on the wire and calculated statistics about network traffic. The application also provides a graphical ping, trace route and DNS lookup functions for added usability in network troubleshooting.

The Network Traffic Monitor captures all visible packets on the network to decode and extract pertinent information for use by the application. It also sends messages to all the devices on the network to retrieve individual data about each host. Collected information is displayed in the form of graphs, tables and charts.
