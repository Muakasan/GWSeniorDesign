GW Senior CS Design 2002

Jamie Wellington: Laban Notation Software

Laban Notation Software:

Laban Notation Software allows a choreographer to electronically produce notation of a dance piece and create an animation of that dance piece. LNS provides a graphical user interface in which the user drags labanotation symbols from a toolbar onto a notation staff. Laban Notation Software has four separate modules: (1) the graphical user interface, (2) the input/output module, (3) the monitor module, and (4) the animation module.
