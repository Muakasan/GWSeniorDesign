GW Senior CS Design 2013

Tyler Kaczmarek: NeuralCam

NeuralCam-An Accessible Webcam Controller : NeuralCam is an accessible system that allows a user with severe motor disabilities to control a mobile webcam. The system consists of an EEG headset, which scans the user's brainwaves to gather input and issue commands, as well as a webcam mounted on a mobile stand, which will receive these commands. The user will issues commands through the EEG headset to control the tilt, pan and zoom of the mobile webcam, which will allow them greater ability to observe their surroundings, which may not have been easily accomplished without such assistance.
