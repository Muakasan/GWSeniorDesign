GW Senior CS Design 2013

Sara Casay: CodeSleuth

CodeSleuth: CodeSleuth allow users to query information about the use of Java from a wide collection of open source Java projects. The user inputs a structural query about the code they are looking for using the query language developed specifically for this project, and either receives the matching code or a number depending on the information requested.
