David Breneisen: Handy Chem

Handy Chem-An Automated Grading Tool For Chemistry:

Handy Chem uses handwriting recognition in order to GRADE chemistry homework and tests. Teachers set up questions with an answer key by writing a structural formula that Handy Chem reads. Then students answer those same questions and Handy Chem matches the teacher's answer key with the student's response.
