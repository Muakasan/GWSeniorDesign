Chris Krawiec: Memory Scheduling

Memory Scheduling-A New Approach: Memory scheduling is a new approach to memory allocation and management that can be applied in environments where the traditional paging approach is not viable. The idea draws inspiration from cpu scheduling and applies a similar mechanism to memory management. Processes can request memory in a fashion similar to the traditional paging approach, however they will be required to aggressively return that memory to the system to prevent the bloat and resource exhaustion often seen under paging.
