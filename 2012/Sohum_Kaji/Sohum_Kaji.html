GW Senior CS Design 2012

Sohum Kaji: Impel

Impel-Law Database Search Analysis Tool: Impel will be used to search law databases. The allure of using Impel is that it will provide enough features such that novices can use the system and will have the simplicity that expert users can take advantage of. These features include using Natural Language Searching to properly understand the user’s query, clustering to make finding results faster, and a law glossary to hone in on certain kinds of documents. Impel will be web-based and will not require the user to download/possess the database or program.
